# NAGO REFORGED API - BACKEND version 0.0.1
ERP para la gestión de productos, invetarios, empleados , nomina y ventas; Ademas, permite la generación de ordenes de producción
ordenes de compra, ordenes de servicios y ordenes de entrega de producto a clientas.
En este solo se manejara la parte con la que interactua los usuarios.

## Project Structure

- ´Enums´ - Carpeta para establecer los enums usado por el sistema.
- ´Schemes´ - Esquemas para la validación de formularios o solicitudes recibidas
- ´Security´ - Archivos para establecer la seguridad del sistema
- ´Controllers´ - En esta carpeta se encontraran los Endpoint usados por el sistema.
- ´Models´ - Aquí se encuentras los modelos de las bases de datos.
- ´Context´ -Aquí está definido el contexto de la base de datos

## Project Info
- Este sistema esta creado usando C# con ASP.NET 9 controller based
- Se usa una base de datos POSTGRESQL Y MONGODB
- Se buscara el despliegue en cloud con AWS.
